# Local AI Deployment & Linux Administration Lab

## Overview
Built and configured a Debian-based Linux environment within ChromeOS 
Crostini to deploy and run a local AI model completely offline using 
Ollama and the Qwen 2.5B language model.

## Environment
- OS: Debian Linux via ChromeOS Crostini
- Hardware: CPU-based inference (no GPU)
- Tools: Ollama, APT, Bash, curl

## Key Tasks Completed
- Configured Linux development sandbox and package management
- Diagnosed and resolved APT dependency and repository conflicts
- Debugged curl, shell syntax, and package installation errors
- Installed and configured Ollama to host Qwen 2.5B locally
- Ran fully offline AI inference via command line

## Skills Demonstrated
Linux Administration | Bash | CLI Troubleshooting | 
APT Package Management | AI Infrastructure | Offline Deployment
